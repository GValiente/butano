bn::affine_bg_attributes_hbe_ptr class
#include <bn_affine_bg_attributes_hbe_ptr.h>

std::shared_ptr like smart pointer that retains shared ownership of a H-Blank effect which changes the attributes to commit to the GBA register of an affine background in each screen horizontal line.

static affine_bg_attributes_hbe_ptr bn::affine_bg_attributes_hbe_ptr::create(affine_bg_ptr bg, const span<const affine_bg_attributes>& attributes_ref)

Creates an affine_bg_attributes_hbe_ptr which changes the attributes to commit to the GBA register of an affine background in each screen horizontal line.

Parameters
bg Affine background to be modified.
attributes_ref Reference to an array of 160 affine_bg_attributes objects with the attributes to commit to the GBA register of the given affine background in each screen horizontal line.
Returns The requested affine_bg_attributes_hbe_ptr.

The attributes are not copied but referenced, so they should outlive the affine_bg_attributes_hbe_ptr to avoid dangling references.
