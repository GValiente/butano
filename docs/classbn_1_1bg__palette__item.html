bn::bg_palette_item class
#include <bn_bg_palette_item.h>

Contains the required information to generate background color palettes.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions
    • Friends

The assets conversion tools generate an object of this type in the build folder for each *.bmp file with regular_bg or affine_bg types.

The colors are not copied but referenced, so they should outlive the bg_palette_item to avoid dangling references.
