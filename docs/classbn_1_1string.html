#include <bn_string.h>
template<int MaxSize>
bn::string class

std::string like container with a fixed size buffer.

Template parameters
MaxSize Maximum number of characters that can be stored.

Contents

  • Reference

It doesn't throw exceptions. Instead, asserts are used to ensure valid usage.
