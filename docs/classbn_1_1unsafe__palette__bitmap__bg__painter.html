bn::unsafe_palette_bitmap_bg_painter class
#include <bn_unsafe_palette_bitmap_bg_painter.h>

Performs low-level painting on palette bitmap backgrounds without validating the input parameters.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Only one bitmap painter can be active at the same time.
