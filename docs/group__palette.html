Display » Color palettes module

Contents

  • Reference
    • Files
    • Namespaces
    • Classes

On the GBA, both sprites and backgrounds have 256 colors organized in blocks of multiples of 16 colors (512 colors in total). Each one of these blocks is called color palette.

There are 8 bits per pixel palettes (which can refer to all colors) and 4 bits per pixel palettes (which can access only to 16 consecutive colors).

Butano color palettes system allows to manage individual 4 bits per pixel palettes for sprites and backgrounds, but it only can be one 8 bits per pixel backgrounds palette and one 8 bits per pixel sprites palette: all 8 bits per pixel backgrounds share the same 8 bits per pixel backgrounds palette and all 8 bits per pixel sprites share the same 8 bits per pixel sprites palette.
